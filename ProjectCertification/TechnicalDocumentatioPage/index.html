Intro

Python is an interpreted, high-level, general-purpose programming language, known for its simplicity and elegance. Created by Guido van Rossum and first released in 1991, Python has since become one of the most popular programming languages in the world.

Its design philosophy emphasizes code readability with the use of significant whitespace and a syntax that allows programmers to express concepts in fewer lines of code than might be used in languages such as C++ or Java.

Python is dynamically typed and garbage-collected, which makes it an excellent choice for a wide range of applications, from simple scripts to complex machine learning algorithms. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming. Python's comprehensive standard library, rich ecosystem of third-party packages, and active community contribute to its versatility, allowing developers to work on web development, data analysis, artificial intelligence, scientific computing, and more.

Variables

In Python, a variable is a name attached to a particular object. Variables allow you to store data, manipulate it, and pass it throughout your program. Unlike some programming languages, Python does not require you to declare the type of your variable upfront; it dynamically infers the type based on the value you assign to it. This feature makes Python highly flexible and user-friendly, especially for beginners.

This is for declaring strings

name = "Alice"

This is for declaring integers:

age = 13

And this is for declaring booleans:

is_student = true

Have you noticed that you don't need to specify the type of the varibale? This is because Python is dynamically typed, which means the type of a variable can change based on the value assigned to it.

Print

Getting Started with the print Command in Python

The print command is one of the most frequently used functions in Python, essential for outputting data to the console. It serves as a powerful tool for displaying strings, numbers, and other objects, making it invaluable for debugging and providing user feedback.

At its simplest, the print command requires just the message or data you wish to display:

print("Hello, World!")
#This command outputs the string Hello, World! to the console.

print can take multiple arguments, separated by commas, and will output them in a single line separated by spaces by default:

print(name, "is", age, "years old.")
#This will output: Alice is 30 years old.

You can customize the separator between items using the sep parameter:

print("Python", "Java", "C++", sep=", ")
#This will output: Python, Java, C++.

By default, print adds a newline character (\n) at the end of the message, so subsequent print calls are on new lines. You can change this behavior with the end parameter:

print("Hello,", end=" ")
print("World!")
#This will output: Hello, World!

The print command can be combined with string formatting methods like f-strings (formatted string literals) for more dynamic outputs:

print(f"{name} is {age} years old.")
#This will also output: Alice is 30 years old.
Cycles

Introduction to Python Loops

Loops in Python are a fundamental concept used to repeat a block of code multiple times. Python provides two primary types of loops: the for loop and the while loop. Understanding how to use these loops allows for efficient repetition of tasks, making code more compact and readable.

The for Loop

The for loop in Python is used to iterate over a sequence (such as a list, tuple, dictionary, set, or string) and execute a block of code for each item in the sequence.

for i in range(5):
   print(i)

This code snippet uses range(5) to create a sequence of numbers from 0 to 4 and prints each number. The for loop iterates through this sequence, assigning each value to i and executing the print statement in each iteration.

The while Loop

The while loop in Python repeatedly executes a block of code as long as a condition is true. It's useful for situations where you need to loop until a certain condition changes, which may not be a fixed number of times.

x = 0
while x < 5:
   print(x)
   x += 1

This snippet starts with x set to 0 and continues to loop as long as x is less than 5. Inside the loop, it prints the current value of x and then increments x by 1. Once x reaches 5, the condition becomes false, and the loop stops.

Breaking Out of a Loop

Sometimes you may want to exit a loop before it has gone through all the items. Python provides the break statement for such situations.

for i in range(10):
   if i == 5:
      break
   print(i)

This code will print numbers from 0 to 4. When i equals 5, the break statement is executed, causing the loop to terminate early, and thus the numbers 5 through 9 are not printed.

Continuing to the Next Iteration

If you want to skip the rest of the code inside the loop for the current iteration and move directly to the next one, you can use the continue statement.

for i in range(10):
   if i % 2 == 0:
      continue
   print(i)

This snippet prints only odd numbers between 0 and 9. If i is even (i.e., i % 2 == 0), the continue statement is executed, which skips the print statement and proceeds to the next iteration of the loop.

Functions

Introduction to Python Functions

Functions in Python are defined blocks of code designed to perform a specific task. They improve code reusability and organization. A function can take parameters, execute code, and return a result. Python functions are defined using the def keyword followed by a function name and parentheses ().

Defining a Function

To define a function in Python, use the def keyword, followed by the function name and parentheses. If your function takes arguments, you would list them inside these parentheses. The code block within every function starts with a colon : and is indented.

def greet(name): print(f"Hello, {name}!")

This code snippet defines a function named greet that takes a single parameter, name, and prints a greeting message using that name.

Calling a Function

After defining a function, you call it by using its name followed by parentheses. If the function requires arguments, you'll include them inside the parentheses.

greet("Alice")

This line calls the greet function defined earlier, passing "Alice" as an argument, which results in the output: Hello, Alice!

Return Values

A function can return a value back to the caller using the return keyword. This is useful when you need the function to produce a result that you want to use later in your program.

def add(a, b): return a + b

This function, add, takes two parameters, a and b, adds them together, and returns the result.

Default Argument Values

You can provide default values for function arguments. If the caller does not provide a value for an argument with a default value, the function will use the default.

def greet(name="World"): print(f"Hello, {name}!")

In this version of the greet function, name has a default value of "World". If greet is called without an argument, it will print "Hello, World!"

Requests

Working with the Python Requests Library

The Python requests library simplifies making HTTP requests to web servers, allowing you to interact with APIs or fetch web page data easily. It abstracts the complexities of making requests behind a beautiful, simple API.

Installing the Requests Library

Before you can start making HTTP requests, you need to install the requests library. If you haven't installed it yet, you can do so using pip:

pip install requests

Importing the Requests Library

Once installed, you can use the requests library by importing it into your Python script.

import requests

Making a GET Request

To retrieve data from a specified resource, use the GET method. The requests.get() function takes the URL of the resource as a parameter.

response = requests.get('http://api.example.com/data') print(response.text)

This code snippet sends a GET request to http://api.example.com/data and prints the response text. The response object contains many properties and methods to inspect the response from the server, such as status_code, text, and json() for JSON responses.

Making a POST Request

To send data to a server to create or update a resource, use the POST method. The requests.post() function requires the URL and the data you wish to send. Data can be passed as a dictionary, which will be automatically form-encoded by the library.

data = {'key': 'value'} response = requests.post('http://api.example.com/submit', data=data) print(response.text)

This snippet sends a POST request to http://api.example.com/submit with some data. The server's response is captured in the response object and printed out. For JSON data, use the json parameter instead of data.

Conclusion

The requests library in Python makes HTTP communication intuitive and straightforward. By following these examples, you can easily integrate GET and POST requests into your Python projects to interact with web services or APIs efficiently.

Data Types

Key Python Data Types

Python offers a variety of data types used to store different kinds of information. Here are six important data types essential for Python programming:

  • Integer (int)

    Represents whole numbers, positive or negative, without decimals. Used for arithmetic operations, loops, and more.

    example = 42

  • Floating Point Number (float)

    Represents real numbers and includes a decimal point. Essential for any calculations requiring fractional numbers.

    example = 3.14

  • String (str)

    A sequence of characters used to store text. Strings are immutable and can be manipulated with various methods.

    example = "Hello, world!"

  • Boolean (bool)

    Represents one of two values: True or False. Booleans are often the result of comparisons or conditions in Python.

    example = True

  • List

    An ordered collection of items that can be of mixed data types. Lists are mutable, allowing modification after creation.

    example = [1, "two", 3.0]

  • Dictionary (dict)

    A collection of key-value pairs. Dictionaries are unordered and mutable, allowing data to be stored and accessed efficiently.

    example = {"name": "Alice", "age": 30}

Each data type has its unique properties and usage scenarios, making Python a versatile and powerful programming language.
